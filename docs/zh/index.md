# UnHacknetJam

一场专注于原版游戏的非官方 Hacknet 创作节。

## 这是什么？

本次创作节是 [FBIK.](https://fbik.top) 个人发起的一项倡议，旨在倡导一种独特的 Hacknet 扩展设计理念：

**强大、创新的体验，同样可以在原版 Hacknet 的约束下打造**

我们相信，真正的创造力源于限制，并在探索（而非赶工）的空间中茁壮成长。

## 目标

鼓励 Hacknet 扩展开发者从一个全新的角度探索创作：

**释放 Hacknet 扩展的潜力。**

## 时间线与参与方式

### 核心活动：为期一个月的创作周期

为期一个月的充裕时间，旨在进行深思熟虑的实验和迭代。重点是**可持续的创作，不受时间压力影响。**

### 持续的创作挑战

在周期开始时公布的**创作主题**将持续开放。我们欢迎并鼓励**在任何时间**（即使在核心活动结束后）受其启发的创作。

## 挑战内容

~~所有参赛作品都受一个特定的**创作主题**启发（可能包括主题、技术或设计上的限制）。该主题将由主办方 ([FBIK.](https://fbik.top)) 在一个月创作窗口开始时公布，并随后永久发布在此处。~~

**Actions使用受到限制！**

- 以下Actions**仅可用作StartingActions**，而且它们的条件必须是 `OnConnect target="playerComp"`：AddAsset, AppendToFile, DeleteFile, CopyAsset
- 以下Actions可自由使用：ShowNode, RunFunction, StartScreenBleedEffect, CancelScreenBleedEffect, SaveGame, HideNode, HideAllNodes, GivePlayerUserAccount

**禁止使用任何模组或工具来改变 Hacknet 的运行方式**

例外情况：

- [fengxu-30338/HacknetChineseSupport](https://github.com/fengxu-30338/HacknetChineseSupport)（本质上是一个使用自定义字体的 Pathfinder 模组），但是你不能启用“多色字体”。
- [fengxu-30338/HacknetFontReplace](https://github.com/fengxu-30338/HacknetFontReplace) HacknetChineseSupport 的新版, 但是一样你不能启用“多色字体”。
- 任何仅用于本地开发辅助，玩家无需使用的工具。（例如，[AutumnRivers/HacknetThemeEditor](https://github.com/AutumnRivers/HacknetThemeEditor)）

## 日期

核心的一个月创作周期目前计划在 **2026年2月** 进行。

_创作挑战本身没有截止日期。_

## 规则

- 你必须严格遵守 **挑战内容**。
- 所有参赛作品必须遵守 [Hacknet 社区行为准则](https://www.hacknet-os.com/Support/CodeOfConduct.txt)。仅使用你有权使用的素材，并尊重他人。
- 鼓励组建团队并协作完成提交。
- 为保持创作节精神的新鲜感，在创作节主题公布前已提交的任何扩展，均不符合参与创作节时间线重点展示的资格。
- 为确保对所有参与者评审过程的公平性和一致性，请不要在一个月活动结束后更新你在创意工坊的提交。如果你希望继续开发你的扩展，我们建议你为更新版本创建一个新的创意工坊项目。
- 请在创意工坊描述中展示你的邮箱。它将用于社区之选亮点的评审。

## 参与方式

### 参与核心一个月活动

若希望作品被纳入创作节时间线重点展示，扩展必须在**一个月窗口期内**上传至 Steam 创意工坊，并标记为 `#UnHacknetJam2026`。

欢迎你在整个月内对你的作品进行迭代。这旨在鼓励探索、打磨和健康的节奏。专注于一个完整、扎实的想法，而不是过于宏大的规模。

### 参与持续的创作挑战

创作主题将无限期开放供解读。
只需在 Steam 创意工坊上使用 `#UnHacknetJam2026` 标签即可，无论何时完成！我们可能会在未来的社区亮点中展示迟交的作品。

## 认可与社区

核心活动结束后，我们将通过两种方式庆祝成果：

- **社区之选亮点：** 一份精选列表，展示获得其他参与者最多积极反馈和游玩次数的扩展。
- **主办方之选亮点：** 一份单独的列表，展示在主办方 ([FBIK.](https://fbik.top)) 看来，最能引人入胜地探索或体现本次创作节核心设计理念的扩展。

目的是展示多元化的方法并激发讨论。

## 评审机制

鼓励参与者对彼此的作品进行评分并提供建设性反馈。

### 社区之选亮点

如果你想为其他作品评分，请从以下几个方面进行评估（以0.5分为增量）：

- **故事 / 叙事** (2.5 分)
- **完成度 / 打磨** (2.5 分)
- **创造力 / 概念** (2.5 分)
- **趣味性 / 享受度** (2.5 分)

在核心活动期间提交社区评分的专用方法，将在临近活动期间或活动期间提供。

请不要忘记在创意工坊描述中展示你的邮箱。它将用于社区之选亮点的评审。

### 主办方之选亮点

主办方之选亮点由 [FBIK.](https://fbik.top) 独力策划，基于 [#这是什么？](#what-is-this) 中描述的哲学契合度。这是一项个人倡议，旨在彰显与其核心理念相符的作品。

### 后续亮点可能性

主办方也可能在整个 **2026年** 期间，对提交给**持续的创作挑战**的优秀作品进行评审。虽然这些迟交的作品不属于核心活动，但杰出的作品**可能会在主办方决定下，在单独的年底亮点中展示**，以庆祝在最初时间框架之外涌现的优秀创意。

## 关于支持的说明

本次创作节是自愿运行的。没有现金奖励。主要的回报是**社区认可、建设性反馈以及在有趣的限制条件下创造新事物的满足感。**

如果未来出现社区赞助或合作伙伴关系以支持创作者，详情将在此透明地沟通。目前的重点纯粹在于技艺、社区和实验的乐趣。
